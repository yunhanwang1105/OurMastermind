How to play

Don't worry if you don't fully understand the rules. Once you start a game, we will provide guides that should be easily to follow. (starter friendly)

The game is played using:

  1. Wait for the code setter set the code. (Setter will have 60 secs to set the code.)
  2. Code guesser now need to take a guess. Make the guess by clicking on a colour. The'undo' button will remove the most recent colour you clicked.
  3. Code setter now responds to the given guess by putting key pegs.
    • One black key peg means the guesser has one correct colour in right position.
    • One white key peg means the guesser has one colour correct but no in right position.
    • If nothing is placed on a key peg position, the guesser then has a wrong colour.
    Notice: The orders don't matter, they only reflects how many correct/incorrect/partially correct colours are there! If there is no pegs, means that one colour is not part of the code. We've designed a logic, if the setter gives incorrect responses, a notification will rise.
  4. Guesser makes the next guess, based on what feedback (s)he received. Step 2-4 are repeated until the game ends.
  5. Game End Conditions:
    • Turns > 10.
    • One player takes too long when it's his/her turn.
    • The guesser worked out the code.
